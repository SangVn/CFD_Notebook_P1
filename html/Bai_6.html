Bài 6. Phương trình Burgers (dòng chảy có nhớt)¶

Phng trnh Burgers dng y : $$\frac {\partial u}{\partial t} + u \frac{\partial u}{\partial x} = \nu \frac{\partial^2 u}{\partial x^2} \qquad (1)$$ y hng t nht c vai tr lm gim bin sng v ngn chn s xut hin kh nng c nhiu nghim (hnh 1).

tm nghim gn ng ca phng trnh ny ta s dng s sai phn phi cho thi gian v sai phn trung tm cho khng gian: $$\frac{u_i^{n+1}-u_i^n}{\Delta t} + u_i^n \frac{u_{i+1}^n - u_{i-1}^n}{2\Delta x} = \nu \frac{u_{i+1}^n - 2u_i^n + u_{i-1}^n}{\Delta x^2}$$

Ta c cng thc xc nh nghim:

$$u_i^{n+1} = u_i^n - u_i^n \frac{\Delta t}{2\Delta x} (u_{i+1}^n - u_{i-1}^n) + \nu \frac{\Delta t}{\Delta x^2}(u_{i+1}^n - 2u_i^n + u_{i-1}^n)$$

Bi ton

Ta s xem qu trnh lan truyn sng xung kch vi cc iu kin: $$u(x, 0) = 1, -1 \leq x \leq 0; \quad u(x, 0) = 0, 0 < x \leq 1$$ $$u(-1, t) = 1, \quad u(1, t) = 0$$ Nghim chnh xc ca bi ton ny c dng: $$u(x, t) = f(\xi) = 1/(1+e^{\xi/2\nu}), \xi = x-t/2$$

Coding

bi ny ta s lm quen vi th vin sympy - h tr tnh ton cc biu thc ton hc biu tng (symbolic). Ci ny ging vi chc nng tng t trong Matlab, Mathematica. bin biu thc ton hc thnh mt hm tnh ton ta s s dng module lambdify.

import sympy as sp # gi th vin sympy

from sympy.utilities.lambdify import lambdify #gi module lambdify
from sympy import init_printing #

init_printing(use_latex=True) #in biu thc dng latex

xi, nu = sp.symbols('xi nu') #khai bo biu tng 
f = 1/(1 + sp.exp(xi/(2*nu)))    #khai bo biu thc f - nghim chnh xc 
fxi = lambdify((xi, nu), f)  #chuyn biu thc f thnh hm fxi vi bin l xi v nu

fdiff = f.diff(xi)  #d dng tnh c o hm fxi theo xi - test

#thng s tnh ton
nx = 201
dx = 2./(nx-1)
x = np.linspace(-1, 1, nx)

#bc thi gian
r = 0.05
dt = r*dx 

#iu kin ban u
u0 = np.zeros(nx)
u0[:int(1./dx)] = 1

#nghim chnh xc, nu -  nht
def u_exact(nu, t):
    xi = x - t/2
    return np.array([fxi(xi_i, nu) for xi_i in xi]) 

#tm nghim vi bin u vo nu, v s bc thi gian nt
def solver(nu, nt):
    u = u0.copy()  #mng u cha nghim
    un = u0.copy() #mng un cha nghim tm thi
    for n in range(nt):
        un = u.copy() #copy gi tr u - lu gi tr  bc hin ti
        #tm nghim  bc thi gian tip theo
        u[1:-1] = un[1:-1] - dt/(2*dx)*un[1:-1]*(un[2:] - un[0:-2]) +\
                  nu*dt/dx**2*(un[2:] - 2*un[1:-1] + un[0:-2])
        u[0] = 1
        u[-1] = 0
    return u

#xt trng hp nu = 0.1
nu = .1
u_1 = solver(nu, 180)
u_2 = solver(nu, 360)
u_3 = solver(nu, 720)
u_ext = u_exact(nu, 720*dt)

plt.plot(x, u0, label='u_0')
plt.plot(x, u_1, label='u_1')
plt.plot(x, u_2, label='u_2')
plt.plot(x, u_3, label='u_3')
plt.plot(x, u_ext, label='u_exact')
plt.legend()
plt.show()
