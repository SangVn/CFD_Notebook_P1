Bài 4. Phương trình Laplace hai chiều¶

Phng trnh Laplace hai chiu l phng trnh o hm ring cp hai dng eliptic: $$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$$ S dng ton t Laplace phng trnh c vit li dng $\Delta u = 0$. Nh cc bn thy trong phng trnh ny khng c o hm ring theo thi gian, iu ny c ngha l nghim u khng ph thuc (khng thay i) theo thi gian: $\partial/\partial t = 0$. Mt trong cc hin tng vt l m phng trnh m t l qu trnh dn nhit n nh, khng c ngun nhit. Trng hp c ngun nhit c m t bng phng trnh Poisson: $\Delta u = f(x,y)$. Khi gii phng trnh Laplace c ba dng iu kin bin:

iu kin bin Diriclet: cho gi tr nhit trn bin

iu kin bin Neumann: cho gi tr o hm hm nhit trn bin, chnh l tc phn tn nhit ra mi trng xung quanh

iu kin bin hn hp: kt hp hai iu kin trn

Bi ton:

  • iu kin ban u: u(x, y) = 0 trn min hnh vung (0,1)x(0,1)
  • iu kin bin hn hp: u(0, y) = 0; u(1, y) = y; $\partial u/\partial y = 0$ ti y = 0, y = 1

S sai phn: gii phng trnh Laplace, ta s dng s 5 im: $$\frac{u_{i+1, j} - 2 u_{i, j} + u_{i-1, j}}{(\Delta x)^2} + \frac{u_{i, j+1} - 2 u_{i, j} + u_{i, j-1}}{(\Delta y)^2} = 0$$ vi sai s xp x $O((\Delta x)^2, (\Delta y)^2)$.

Do khng c thi gian nn phng php gii phng trnh Laplace khng ging nh nhng g ta lm cc bi trc. C nhiu phng php khc nhau c p dng. y ta s dng phng php lp nh sau:

  • a phng trnh v dng: $$u_{ij} = \frac{(\Delta y)^2 (u_{i+1, j} + u_{i-1, j}) + (\Delta x)^2 (u_{i, j+1} + u_{i, j-1})}{2((\Delta x)^2 + (\Delta y)^2)} $$
  • thay iu kin ban u, iu kin bin vo phng trnh trn tm $u_{i, j}$ bc lp th nht.
  • s dng kt qu tm c lm d liu u vo tm $u_{i, j}$ bc tip theo
  • lp li bc trn cho ti khi nghim tm c hai bc lin tip sai khc rt nh, khi ta ni nghim hi t. Nu ta lp qu trnh ny N bc, th vi N cng ln nghim cng chnh xc. Hoc ta c th kim tra iu kin sau (cn c gi l chun) dng qu trnh lp:
$$\frac {\sum (|u_{i, j}^{sau}| - |u_{i,j}^{trc}|)}{\sum |u_{ij}^{trc}|} < \varepsilon$$

Coding:

  • Vi bi ton hai chiu (x, y) ta coi nh ang lm vic vi mt chiu x ti cc gi tr y c nh
  • cha gi tr u ta khai bo mng hai chiu gm ny hng, nx ct. u[j, i] - phn t hng j (trc y), ct i (trc x)
  • biu din gi tr u(x, y) ta s s dng th vin mplot3d, xy dng hm plot(u)
  • tm nghim ta xy dng hm solver(u, y, epsilon)

import numpy as np
import matplotlib.pyplot as plt
from matplotlib import cm #colormap
from mpl_toolkits.mplot3d import Axes3D #khai bo th vin  v  th 3D

#chia li
nx = 21
ny = 21
dx = 1./(nx-1)
dy = 1./(ny-1)

x = np.linspace(0, 1, nx)
y = np.linspace(0, 1, ny)

#iu kin ban u
u = np.zeros((ny, nx)) # to mng ny hng, nx ct, gi tr mi phn t u[j, i] = 0

#iu kin bin
u[:, 0] = 0 #u = 0, x = 0
u[:, -1] = y #u = y, x = 1
u[0, :] = u[1, :] #du/dy = 0, y = 0
u[-1, :] = u[-2, :] #du/dy = 0, y = 1

#khai bo hm plot vi d liu u vo l gi tr hm u
def plot(u):
    fig = plt.figure() #Top level container for all plot elements.
    ax = fig.gca(projection='3d') #Get the current Axes
    X, Y = np.meshgrid(x, y) # to li im (X, Y)
    surf = ax.plot_surface(X, Y, u, cmap=cm.plasma)
    ax.set_xlim(0, 1)
    ax.set_ylim(0, 1)
    ax.set_xlabel('x')
    ax.set_ylabel('y')
    ax.set_zlabel('u')
    ax.view_init(30, 250)
    #fig.colorbar(surf)
    plt.show()
    
#biu din iu kin ban u, iu kin bin
plot(u)

#hm solver vi cc d liu u vo cn thit: u, y, epsilon

def solver(u, y, epsilon):
    un = u.copy()           #khai bo mng un cha gi tr tm thi
    norm = 2*epsilon        #gn gi tr chun ban u > epsilon 
    while (norm > epsilon): #nu norm <= epsilon th thot khi vng lp
        un = u.copy()       #lu gi tr hin ti
        
        #tnh u[j, i]  bc tip theo
        u[1:-1, 1:-1] = (dy**2*(un[1:-1, 2:] + un[1:-1, 0:-2]) + dx**2*(un[2:, 1:-1] + un[0:-2, 1:-1]))/\
                        (2*(dx**2 + dy**2))
        #iu kin bin
        u[:, 0] = 0         #u = 0, x = 0
        u[:, -1] = y        #u = 1, x = 1
        u[0, :] = u[1, :]   #du/dy = 0, y = 0
        u[-1, :] = u[-2, :] #du/dy = 0, y = 1
        
        #tnh chun 
        norm = np.sum(np.abs(u[:]) - np.abs(un[:]))/np.sum(np.abs(un[:]))
        
    return u
